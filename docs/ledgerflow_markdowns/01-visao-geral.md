# LedgerFlow — Visão Geral

## Objetivo

LedgerFlow é uma plataforma de gestão financeira pessoal e profissional construída para demonstrar arquitetura enterprise moderna utilizando:

- C# + ASP.NET Core
- Angular
- PostgreSQL
- Redis
- Hangfire

O projeto terá evolução incremental, começando como um monolito modular bem estruturado e evoluindo para cenários enterprise.
